# supportSignal
Level 3 Diagnostic Agent for Openshift AI, which can be called by CEE BI First responder, which then will analyze must-gather, search through database/KCS of Red Hat and provide suspected issues with action plan. 

The CEE First Responder is designed as horizontal solution. The Support Signal has potential to be the vertical specialist in individual products. 

Red Hat OpenShift AI (RHOAI) stack involves Kernel -> GPU Driver -> Nvidia Operator -> RHOAI Operator -> Notebook Controller. 

This is meant to be L3 level expert to analyze and diagnose the case in continuous fashion for OpenShift AI. The subsequent case updates or latest must-gather can be processed as well and new diagnosis can be performed.
